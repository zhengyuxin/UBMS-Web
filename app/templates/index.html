Complete Support Library of Binary Components and Drivers

Intel® UBMS effectively eliminates the need for source code and compiler environment in the legacy embedded development. Its advanced user-mode supports detailed configuration and creation of firmware components for custom hardware and allows optimized firmware image for multiple feature sets. Based on this kind of framework, the 3rd party hardware device vendor can directly publish their latest device driver in binary format, which largely helps to protect their intellectual property and competitive advantage.
